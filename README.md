# zyb-appstore-iot-admin(作业帮应用商店代理)
## 功能特性
完整的应用管理（增删改查）
多级权限系统（Super Admin / Manager）
SN码白名单控制
Cloudflare R2 存储支持
1GB 大文件上传支持
完整的 API 接口
## API 文档
GET /iot-study/appStore/apps - 获取应用列表
GET /iot-study/appStore/apk - 获取应用详情
POST /manage_app_data_zybiot_1223/add - 添加应用（需要认证）
## 管理后台
超级管理员：/super_admin_config_1223
普通管理员：/manage_app_data_zybiot_1223
